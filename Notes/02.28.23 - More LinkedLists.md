## Temporary Pointers

```python
def zipper_2(head1, head2):
    temp1 = head1
    temp2 = head2

    while temp1 and temp2:
        next1 = temp1.next
        next2 = temp2.next

        temp1.next = temp2
        temp2.next = next1

        temp1 = next1
        temp2 = next2

    return head1
```
